    setTitle

    • setTitle(
          title: string | RawMessage | (string | RawMessage)[],
          options?: TitleDisplayOptions,
      ): void

      Parameters

      • title: string | RawMessage | (string | RawMessage)[]
      • Optionaloptions: TitleDisplayOptions

      Returns void

      Remarks

      Will cause a title to show up on the player's on screen display. Will clear the title if set to empty string. You can optionally specify an additional subtitle as well as fade in, stay and fade out times.

      This function can't be called in read-only mode.

      Throws

      This function can throw errors.

      Example: countdown.ts

        let players = mc.world.getPlayers();

        players[0].onScreenDisplay.setTitle("Get ready!", {
          stayDuration: 220,
          fadeInDuration: 2,
          fadeOutDuration: 4,
          subtitle: "10",
        });

        let countdown = 10;

        let intervalId = mc.system.runInterval(() => {
          countdown--;
          players[0].onScreenDisplay.updateSubtitle(countdown.toString());

          if (countdown == 0) {
            mc.system.clearRun(intervalId);
          }
        }, 20);

      Example: setTitle.ts

        let players = mc.world.getPlayers();

        players[0].onScreenDisplay.setTitle("§o§6Fancy Title§r");

      Example: setTitleAndSubtitle.ts

        let players = mc.world.getPlayers();

        players[0].onScreenDisplay.setTitle("Chapter 1", {
          stayDuration: 100,
          fadeInDuration: 2,
          fadeOutDuration: 4,
          subtitle: "Trouble in Block Town",
        });
