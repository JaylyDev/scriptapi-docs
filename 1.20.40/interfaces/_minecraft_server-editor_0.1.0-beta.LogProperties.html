Interface LogPropertiesBeta

A properties class for the global instance of the logger object. While the logger object is available through the @minecraft/server-editor-bindings.ExtensionContext - using the global instance allows the creator to use this properties class to perform direct server->client messaging and broadcasts.

interface LogProperties {
    player?: Player;
    tags?: string[];
