Class EntityMovementSwayComponentBeta

When added, this move control causes the mob to sway side to side giving the impression it is swimming.

Example: entityMovementSway.js

import { world } from "@minecraft/server";

world.afterEvents.entitySpawn.subscribe(({ entity }) => {
    const movementSway = entity.getComponent("movement.sway");
    movementSway.swayAmplitude;
    movementSway.swayFrequency;
    movementSway.isValid();
});
